Class: Paginator

module:classes/Paginator~ Paginator

new Paginator(pageFormatLabel, pageOrientation, doc)

Paginator is the page manager
Parameters:
Name Type Description
pageFormatLabel string The label of the paper format for all pages. For example, 'A4'
pageOrientation string The label of the orientation for all pages. May be 'portait' or 'landscape'
doc DOMDocument The document given by editor.getDoc() of the tinymce API
